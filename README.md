# App IBank Build

Java-приложение интернет-банка.

## Цель проекта
Разработать backend для системы интернет-банкинга с базовыми банковскими операциями.

## Задачи
- Реализовать REST API для банковских операций
- Настроить Spring Boot приложение
- Подключить базу данных
- Реализовать обработку запросов

## Результат
Работающее Spring Boot приложение с REST API для банковских операций.

## Стек технологий
- Java, Spring Boot
- Maven
- PostgreSQL
